# 8INF853 - Dotnet Core

Un projet Dotnet Core développé dans le cadre du cours d'architecture des applications d'entreprises de l'UQAC.

## Installation

Clonez le projet dans Visual Studio.
:warning: version Windows conseilée

## Base de données

Automatiquement gérée par Visual Studio quand le projet est récupéré. 
Les migrations sont actives et le fichier `Data/Seed.cs` initialise la base de donnée la première fois.

## Stack Technique

> Projet
* ASP Dotnet Core - 2.1
* Microsoft.AspNetCore.Razor.Design - 2.1.2
* BuildBundlerMinifier - 2.9
* Bootstrap - 3.4

> Outils de développement
* Git - 2.21
* Visual Studio Community Edition 2019 - 16.0.1
